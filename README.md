
# Primeiros Passos em Python 🐍

Este repositório contém os meus primeiros scripts desenvolvidos em Python. O objetivo principal foi exercitar a lógica de programação, o uso de operadores aritméticos e estruturas condicionais.

### 🚀 Tecnologias e Conceitos Aplicados:
- **Entrada e Saída de Dados:** Uso de `input()` e `print()` com formatação (f-strings).
- **Aritmética:** Cálculos de sucessor/antecessor, dobro, triplo, raiz quadrada e média.
- **Lógica Financeira:** Cálculo de descontos (5%) e aumento salarial (15%).
- **Conversão de Unidades:** Metros para centímetros/milímetros e conversão de Moeda (Real para Dólar).
- **Estruturas Condicionais:** Verificação de aprovação escolar utilizando `if/else`.
- **Geometria Básica:** Cálculo de área (M²) e consumo de tinta.

### 🎓 Contexto:
Projeto desenvolvido durante a minha graduação em **Ciência da Computação na FAM** e como parte dos exercícios práticos do **Bootcamp Santander**.
